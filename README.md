# Assignment-06

This project is a simple product and cart based web application that I have built using React. The main idea of this project was to understand how to manage data, design UI, and implement basic cart functionality.

## Project Description

In this project, I created a product section where users can see different items with details like name, price, and features. There is also a cart system where users can add products, remove them, and see the total price.

I tried to follow a clean design based on the given Figma layout and make the website responsive for different devices.

## Technologies I Used

- React.js
- JavaScript (ES6)
- Tailwind CSS
- DaisyUI
- React Toastify

## Key Features

- Users can view products in a clean card layout
- Add to cart functionality with instant feedback
- Remove products from the cart
- Dynamic cart count shown in the navbar
- Total price calculation
- Toggle between product and cart section
- Checkout button clears all cart items
- Toast notification for actions (add, remove, checkout)

## Project Structure (Simple Idea)

I tried to keep the project organized:

- components folder for reusable parts like Navbar, Product Card
- data folder for product JSON data
- main App.jsx for handling logic

## Responsive Design

I made the website responsive so that it works on mobile, tablet, and desktop. I used Tailwind CSS classes for this.
